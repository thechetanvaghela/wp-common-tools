== Description ==

The plugin provides common functionality to the site. A site administrator can add page scroll progress information, back to top buttons, hide the admin bar from the front end, change the wp-admin logo and link, and add a page loader to the site.


### Features And Options:
* Page Scroll Progress is displayed at the bottom of the page or as a bar on the top.
* Back to top button
* Hide Admin bar
* Change wp-admin wordpress logo image and link
* SVG support option for upload
* Page loader

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-common-tools` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "Common Tools for Site" plugin through the 'Plugins' screen in WordPress
3. Click on "Common Tools" on the dashboard.
